About Animals CRUD

Welcome to Animals CRUD, a comprehensive web application for managing animal data. This project demonstrates full-stack development using modern web technologies.

Project Overview

This application allows users to perform complete CRUD (Create, Read, Update, Delete) operations on three types of animals: cats, dogs, and mice. The system includes user authentication, ensuring that only logged-in users can modify or delete animal records.

Technologies Used

  • Backend: Node.js with Express.js framework
  • Database: MySQL with connection pooling
  • Authentication: Express sessions with bcrypt
  • Frontend: HTML5, CSS3, Vanilla JS

Features

  • User registration and login system
  • Role-based access control
  • Responsive design "100%" width
  • Real-time search and filtering
  • Statistics dashboard
