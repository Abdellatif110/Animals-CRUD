About Animals CRUD

Welcome to Animals CRUD, a comprehensive web application for managing animal data. This project demonstrates full-stack development using modern web technologies.

Project Overview

This application allows users to perform complete CRUD (Create, Read, Update, Delete) operations on three types of animals: cats, dogs, and mice. The system includes user authentication, ensuring that only logged-in users can modify or delete animal records.

Technologies Used

  • Backend: Node.js with Express.js framework
  • Database: MySQL with connection pooling
  • Authentication: Express sessions with bcrypt password hashing
  • Frontend: HTML5, CSS3, and vanilla JavaScript
  • Styling: Custom CSS with Font Awesome icons

Features

  • User registration and login system
  • Role-based access control (authenticated users can edit/delete)
  • Responsive design for mobile and desktop
  • Real-time search and filtering by tags
  • Pagination for large datasets
  • Statistics dashboard
  • Demo data loading

Database Structure

The application uses four main tables:

  • cats - Stores cat information
  • dogs - Stores dog information
  • mouses - Stores mouse information
  • users - Stores user accounts

Each animal record includes: ID, name, tags, description, image URL, and creation timestamp.

API Endpoints

The application provides a RESTful API with endpoints for all CRUD operations, statistics, and user management.
